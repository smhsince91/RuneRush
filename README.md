This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-onchain`](https://www.npmjs.com/package/create-onchain).


## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Next, run the local development server:

```bash
powershell -ExecutionPolicy Bypass -File .\scripts\dev-local.ps1
```

Open [http://127.0.0.1:3100](http://127.0.0.1:3100) with your browser to see the result.

For phone or LAN testing, run:

```bash
powershell -ExecutionPolicy Bypass -File .\scripts\dev-local.ps1 -Lan
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
